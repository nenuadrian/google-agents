Gemini CLI: What & Why

Gemini CLI is a command-line interface for interacting with Gemini models directly from your terminal. It turns LLM interactions into shell-friendly primitives, so you can:

  • Create a GEMINI.md file to fine-tune goals
  • Generate and refine code inline while you hack
  • Summarise or transform files without leaving the shell
  • Wire Gemini into scripts, Makefiles, and small tools
Typical workflow
  1. Install the CLI and authenticate with your Google account
  2. Set your default model and project
  3. Use simple commands like gemini ask or gemini generate
  4. Understand MCP extensions


Gemini CLI GitHub Repository Gemini CLI Architecture Gemini CLI MCP
Example commands 
# Ask a quick question
gemini ask "Explain Gemini CLI in one paragraph"

# Summarise a file
gemini file summarize README.md

# Generate starter code
gemini code "Python script for a text-based quiz game"

# Enter full mode
gemini

During the workshop we’ll use commands like these as building blocks for the project ideas below.

Proposed Projects

To make things concrete, here are three project ideas you can build during the session. They’re intentionally small but composable, so you can extend them after the workshop.

1. Gemini Quiz Adventure (Game)

A terminal-based story and quiz game. Gemini generates narrative branches, riddles, and adaptive hints based on the player’s answers.

  • Use Gemini CLI to generate questions and story events
  • Track player state (score, progress, difficulty)
  • Optionally log game transcripts for later analysis

Stretch goal: add a “lore mode” where Gemini expands the world based on previous runs.

2. Project & Docs Copilot

A CLI helper that lets you ask questions about local project files and documentation.

  • Command to select a folder / repo
  • Use Gemini CLI to summarise and explain key files
  • Simple Q&A mode: ./docbot "How do I run tests?"

Stretch goal: auto-generate onboarding docs for new contributors.

3. Lightweight Code Reviewer

A small script that sends code snippets or diffs to Gemini and returns focused review comments.

  • Command to pipe git diff into Gemini CLI
  • Configurable “review profile” (strict, fast, beginner-friendly)
  • Optional JSON output for integration with CI

Stretch goal: generate suggested patches you can apply directly.
